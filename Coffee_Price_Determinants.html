Coffee Price Determinants (Updated)

Advanced econometric analysis uncovering the key drivers of global coffee prices, 1994–2022.

Explore New Findings ↓

Project Overview

This project investigates the determinants of global coffee prices using advanced regression modeling, stepwise selection, ANOVA comparisons, and residual diagnostics. The analysis compares models with and without a time trend (Year) to balance predictive power and interpretability.

Key Methods & Tools

  • Data cleaning and transformation (R: dplyr, readxl)
  • OLS regression and stepwise selection
  • Model diagnostics: ANOVA, residual vs. fitted plots, Q-Q plots
  • Variable importance analysis (relaimpo package)

Main Findings

  • Key drivers: Imports (−), Exchange rate (−), World Population (−), Domestic Consumption (+, borderline)
  • Adding Year significantly improves fit (ANOVA p = 0.022), capturing time trends
  • With Year: R² = 0.87; Without Year: R² = 0.83 → stronger residual stability with Year
  • Residuals approximately normal; Q-Q plots show slight tail deviation but acceptable fit

Model Diagnostics and Insights

Residuals vs. Fitted Plots

These plots compare residual scatter with and without Year. Including Year reduces clustering and improves error spread.

Normal Q-Q Plots

Q-Q plots for both models show acceptable normality, with minor tail deviations. Including Year slightly improves residual normality.

Sample Code (R)

# Compare models with and without Year
model_with_year <- lm(`Coffee Price` ~ Year + `Exchange rate` + `World Population` + `Domestic Consumption` + Imports, data = train)
model_without_year <- lm(`Coffee Price` ~ `Exchange rate` + `World Population` + `Domestic Consumption` + Imports, data = train)

# ANOVA comparison
anova(model_without_year, model_with_year)

# Variable importance (relaimpo)
library(relaimpo)
calc_relimp(model_with_year, type = "lmg", rela = TRUE)
