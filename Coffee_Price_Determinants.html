Coffee Price Determinants

Econometric Analysis Using R

Project Overview

This project analyzes the key economic determinants of global coffee prices using econometric techniques. Using a dataset covering 1994‚Äì2022, I applied log-linear regression models, robust standard errors, residual diagnostics, and stepwise selection to identify the most influential factors on coffee price dynamics.

Key Methods & Tools

  • Data cleaning and transformation in R (dplyr, readxl)
  • OLS regression with robust standard errors
  • Model selection with stepwise AIC
  • Diagnostics: residual plots, Q-Q plots, VIF checks
