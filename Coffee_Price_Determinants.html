Coffee Price Determinants

Econometric Analysis Using R

Project Overview

This project analyzes the key economic determinants of global coffee prices using econometric techniques. Using a dataset covering 1994–2022, I applied log-linear regression models, robust standard errors, residual diagnostics, and stepwise selection to identify the most influential factors on coffee price dynamics.

Key Methods & Tools

  • Data cleaning and transformation in R (dplyr, readxl)
  • OLS regression with robust standard errors
  • Model selection with stepwise AIC
  • Diagnostics: residual plots, Q-Q plots, VIF checks

Main Findings

  • Significant drivers: Coffee consumption (+), imports (−), exchange rate (−)
  • High R² (~86%) showing strong model fit
  • Residuals approximately normal, no major violations of assumptions

Residual Diagnostics: Q-Q Plot

This Q-Q plot shows the residual distribution and assesses normality assumptions in the regression model.

Sample Code

model <- lm(ln_CoffeePrice ~ ln_CoffeeConsumption + ln_Imports + ln_ExchangeRate + ln_OilPrices, data = data)
summary(model)
coeftest(model, vcov = vcovHC(model, type = "HC1"))

Takeaways

This project highlights the value of econometric analysis in understanding commodity markets. The skills applied—data wrangling, regression modeling, and diagnostics—are highly transferable to other economic and business contexts.
