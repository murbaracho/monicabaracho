NHS Resource Utilisation & Missed Appointments

Data-driven insights into demand, missed appointments, and public sentiment

Overview

This project explores 30 months of GP appointment data from NHS England. Using structured appointment data and external Twitter data, the analysis uncovers trends, system bottlenecks, and missed appointment risks, guiding future NHS capacity and communication strategies.

Challenge

The NHS faces mounting pressure from increased demand and missed appointments. We sought to answer:

  • Has capacity kept pace with demand?
  • What predicts missed appointments?
  • What does public sentiment reveal about service perception?

Tools Used

Python: Pandas, Seaborn, Scikit-learn, Matplotlib
Natural Language: VADER for sentiment analysis
Environment: Jupyter Notebook, GitHub, HTML export

Actions Taken

  • Explored and visualised appointment trends by region, type, and duration
  • Modelled DNA risk using logistic regression
  • Mapped overutilised vs underutilised ICBs
  • Analysed Twitter hashtags and sentiment using VADER

Insights

  • Overcapacity: NHS ran above 110% capacity on average
  • Missed Appointments: Driven by short booking windows and remote formats
  • Regional Inequality: Urban ICBs had less access per capita than rural ones
  • Public Sentiment: Focused on #job, #digitalhealth, #nhscrisis

💡 Recommendations

  • 🔔 Send automated reminders for short-notice bookings
  • 🧑‍⚕️ Expand locum pools and dynamic staffing for high-volume months
  • 📍 Direct funding to underserved urban regions
  • 📊 Improve metadata (e.g. patient profiles) for predictive models
  • 📡 Track Twitter sentiment weekly to spot reputational risks
